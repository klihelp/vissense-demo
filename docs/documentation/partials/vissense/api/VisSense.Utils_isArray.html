isArray ( function in module VisSense.Utils )

Description

Checks if value is classified as an Array object

VisSense.Utils.isArray([1, 2, 3]);
// => true

(function() { return VisSense.Utils.isArray(arguments); })();
// => false
From lodash: isFunction

Parameters

  • value – {*} –

    The value to check.

Returns

{boolean} –

Returns true if value is an Array , else false .
