onVisible ( method in module VisSense.VisMon )

Description

Fires when element becomes visible

Fires when visibility changes and and state transits from: HIDDEN => VISIBLE HIDDEN => FULLYVISIBLE Fires NOT when state transits from: VISIBLE => FULLYVISIBLE or FULLY_VISIBLE => VISIBLE

VisSense(document.getElementById('example1')).monitor().onVisible(function() {
  Animations.startAnimation();
});

Parameters

  • callback – {Function} –

    A callback function

Returns

{Function} –

A function when called will unregister the callback
