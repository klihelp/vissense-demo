isVisibleByStyling ( function in module VisSense.Utils )

Description

Checks if the element is visible by its style.

var element = document.getElementById('myElement');
VisSense.Utils.isVisibleByStyling(element);
// => true

Parameters

  • element – {DOMElement} –

    A DOM element

Returns

{Boolean} –

True if the element is visible by style and the style of its parents.
