isElement ( function in module VisSense.Utils )

Description

Checks if value is a DOM Element.

var elem = document.getElementById('myElement')
VisSense.Utils.isElement(elem);
// => true

VisSense.Utils.isElement(document);
// => false

VisSense.Utils.isElement(document.body);
// => true

Parameters

  • value – {*} –

    The value to check.

Returns

{boolean} –

Returns true if the given value is a DOM Element, else false.
