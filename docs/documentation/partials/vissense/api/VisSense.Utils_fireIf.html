fireIf ( function in module VisSense.Utils )

Description

A function that when executed runs the given callback only if when evaluates to a truthy value.

var a = 6;
VisSense.Utils.fireIf(function() { return a % 2 === 0; }, function() {
  return a % 3 === 0;
})();
// => true

Parameters

  • when – {Function|Boolean} –

    A function or any other value

  • callback – {*} –

    A function to run if when evaluates to a truthy value

Returns

{Function} –

Returns a function that runs the given callback only if when evaluates to a truthy value.
