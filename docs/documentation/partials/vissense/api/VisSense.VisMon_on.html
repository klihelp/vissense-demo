on ( method in module VisSense.VisMon )

Description

Binds a callback to specific event. Valid events are: - ´update´ - ´hidden´ - ´visible´ - ´fullyvisible´ - ´percentagechange´ - ´visibilitychange´

var monitor = VisSense(...).monitor(...);
       monitor.on('fullyvisible', function() {
  Animations.startAnimation();
});

monitor.on('percentagechange', function(newValue) {
  if(newValue < 0.8) {
    Animations.stopAnimation();
  }
});

monitor.start();

Parameters

  • eventName – {String} –

    The name of the event to bind the callback to

  • callback – {Function} –

    A callback function

Returns

{Function} –

A function when called will unregister the callback
