isFunction ( function in module VisSense.Utils )

Description

Checks if value is classified as a Function object.

VisSense.Utils.isFunction(VisSense);
// => true

VisSense.Utils.isFunction(/abc/);
// => false

    The value to check.

Returns

{boolean} –

Returns true if the value is a function, else false.
