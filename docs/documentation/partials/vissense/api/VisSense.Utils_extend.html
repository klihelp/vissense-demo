extend ( function in module VisSense.Utils )

Description

Overwrites all properties of the destination object with the source object's properties. You can provide an optional callback function to modify the behaviour and/or to manipulate the return value.

VisSense.Utils.extend({ name: 'Max', age: 31 }, { name: 'Bradley', gender: 'male' });
// => { name: 'Bradley', age: 31, gender: 'male' }


VisSense.Utils.extend({
  name: 'Max',
  age: 31
}, {
  name: 'Bradley',
  gender: 'male'
}, function(destValue, srcValue, key) {
  if(key === 'age') return destValue + 42;
  return srcValue;
});
// => { name: 'Bradley', age: 73, gender: 'male' }

Parameters

  • dest – {Object} –

    The destination object.

  • source – {Object} –

    The source object.

  • [callback] – {Function} –

    The function to customize assigning values.
