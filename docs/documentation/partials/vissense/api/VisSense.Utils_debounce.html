debounce ( function in module VisSense.Utils )

Description

Returns a function that delays invoking fn until after delay milliseconds have elapsed since the last time it was invoked.

window.addEventListener('resize', VisSense.Utils.debounce(50, function() { 
  console.log('resizing...'); 
}));
// => logs 'scrolling...' at most every 50ms while the user resizes the browser window

Parameters

  • fn – {*} –

    The function to debounce.

  • delay – {*} –

    The number of milliseconds to delay.

Returns

{Number} –

Returns a function that delays invoking fn until after delay milliseconds have elapsed since the last time it was invoked.
