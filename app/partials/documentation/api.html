vissense

vis.percentage();
gets the current visible percentage (0..1)
vis.isHidden();
´true´ if element is hidden
vis.isVisible();
´true´ if element is visible
vis.isFullyVisible();
´true´ if element is fully visible
vis.state();
gets an object representing the current state
{ "code": 1, "state": "visible", "percentage": 0.55, "visible": true, "fullyvisible": false, "hidden": false "previous" : {} }

vissense-monitor

  • start() starts observing the element
  • stop() stops observing the element
  • update() manually run the update procedure. this will fire any registered handlers
vismon.on();
registers an event handler
vismon.on('update', function(monitor) { ... }); vismon.on('hidden', function(monitor) { ... }); vismon.on('visible', function(monitor) { ... }); vismon.on('fullyvisible', function(monitor) { ... }); vismon.on('percentagechange', function(newValue, oldValue, monitor) { ... }); vismon.on('visibilitychange', function(monitor) { ... });
vismon.state();
gets an object representing the current state
{ "code": 1, "state": "visible", "percentage": 0.55, "visible": true, "fullyvisible": false, "hidden": false "previous" : { "code": 2, "state": "fullyvisible", "percentage": 1, "visible": true, "fullyvisible": true, "hidden": false } }
